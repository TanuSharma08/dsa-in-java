# 🚀 Getting Started

## Prerequisites

- Java 8+
- IntelliJ IDEA / Eclipse / VS Code

---

## Clone the Repository

```bash
git clone https://github.com/TanuSharma08/dsa-in-java.git
```

```bash
cd dsa-in-java
```

---

## Compile & Run

```bash
javac src/arrays/TwoSum.java
```

```bash
java src/arrays/TwoSum
```
